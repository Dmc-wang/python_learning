任务 2：日志分析脚本
（E3)
目标：
    从 log 文件中提取所有 IP 地址，统计出现频率。
常见 log 示例行：
    2023-07-21 10:00:01 - INFO - Connection from 192.168.1.15 port 443
    2023-07-21 10:00:05 - ERROR - Connection timeout from 10.0.0.2
功能要求
    从文件中提取所有 IPv4
    用正则表达式（工程必备）
    提供异常处理
    输出统计结果到一个新文件（如 ip_report.txt）

🛣️ 学习路线（阶段 × 产出 × 用时）
阶段一：Python 基础（约 5 天）

学习内容：
变量、类型、输入输出、流程控制、代码风格（PEP 8）

产出物：

3 个示例程序

10 个基础语法练习

熟悉 Python 运行与 VSCode 环境

阶段二：数据结构与函数（约 7 天）

学习内容：
列表、字典、集合、元组
函数定义、返回值、参数类型、lambda 表达式

产出物：

每种数据结构至少编写 3 个练习

完成 5 个函数编程小题

阶段三：面向对象 OOP（约 6 天）

学习内容：
类、对象、继承、多态、魔法方法、模块与包

产出物：

4 个 OOP 示例程序

一个简单类设计练习（如：学生管理系统）

阶段四：应用与实战项目（约 10–15 天）

学习内容：
文件操作、异常处理、虚拟环境、标准库（os、datetime、random、sys 等）

产出物（可展示项目）：

✔️ 项目 1：命令行 ToDo 工具

argparse 实现命令行参数

JSON / 文件存储任务列表

✔️ 项目 2：文件整理助手

扫描文件夹

根据文件类型自动分类至对应目录

✔️ 项目 3：简单爬虫

requests 获取网页

BeautifulSoup 提取标题 / 链接

数据输出到 CSV

## 📖 项目简介

这是一个集**网络爬虫、数据清洗、数据分析**于一体的实战项目。通过爬取豆瓣电影Top250数据，使用Pandas进行深度分析，帮助你掌握Python数据处理核心技能。

**项目价值**：爬虫 + 数据处理是AI和数据科学项目的基础，本案例贴近实际工作场景。

---

## ✨ 功能特性

- 🔍 **爬虫模块**：自动爬取豆瓣Top250电影数据（支持异常处理和重试）
- 💾 **数据存储**：支持CSV/JSON/Excel多种格式导出
- 🧹 **数据清洗**：Pandas全流程清洗（类型转换、缺失值处理、去重等）
- 📊 **数据分析**：统计报告、相关性分析、文本预处理
- 🧪 **单元测试**：pytest测试覆盖核心功能
- 🚀 **CI/CD**：GitHub Actions自动测试

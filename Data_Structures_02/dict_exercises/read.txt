2. 字典练习题
D1：统计字符串中每个字符出现的次数
    例如 "apple" → { "a":1, "p":2, "l":1, "e":1 }

D2：学生成绩字典处理
    输入：{"Tom":90, "Bob":80, "Alice":95}
    输出最高分学生姓名与分数

D3：交换字典的 key 和 value
    {"a":1, "b":2} → {1:"a", 2:"b"}

D4（应用）：把一个英文句子按单词频率统计
    输入："hello world hello" → 输出词频字典

D5：模拟一个“学生成绩登记管理系统”：
    添加学生和他的成绩
    删除学生和他的成绩
    查询学生和他的成绩
